Programowanie

Języki programowania

Java

Jest to najpopularniejszy język programowania, możmy dzięki niemu stworzyć aplikacje desktopowe, serwisy inernetowe, alpikacje na androida. Charakterycuje obiektowośc, przenośność na różne platformy, dziedziczenie, niezależność od architektury, sieciowośc i obsługa programowania rozproszonego, niezawodność i bezpieczeństwo. Najlpopularniejszy framework javy to Spring.

Javascript

Jest to język stosowany na stronach internetowych po stronie front-endu. Odpowiada on tam za logikę strony internetowej i pozwala na interakcje z użytkownikiem. Najpopularniejsza biblioteka javascriptu to JQuery.

PHP

Czyli interpretowany skryptowy język programowania zaprojektowany do generowania stron internetowych i budowania aplikacji webowych. Mówiąć prościej używa się go po stronie back-endu, gdzie odpowiada za logikę aplikacji po stronie serwera, są w nim napisane całe mechanizmy zaawansowanych stron internetowych lub też inaczej mówiąc serwisów. Stale się rozwija.

SQL

Język ten pozwala nam tworzyć, komunikować się i wykonywać wszelkie operacje na bazach danych. Wykorzustują go inne języki np. java aby komunikować się z bazą danych i pobierać do tworzonej aplikacji dane oraz je tam zapisywać. Jest cała rodzina języków sql np. MySQL, HSQL, SQLite, PostgreSQL.

C#

Charakteryzuje się tym, że jest podobieństwem języka Java. Stworzony przez firmę Microsoft. Ma te same cechy co Java. Najpopularniejszy framework to .Net. Nieustannie się rozwija. Jest często stosowany do produkcji gier np na platformie Unity.

C++

Język programowania ogólnego przeznaczenia, stosowany bardzo często do produkcji gier. Jest następcą języka C i poprzednikiem języka C#. Nie jest językiem zupełnie obiektowym ale pozwala tworzyć obiektowość, wykorzystuje programowanie proceduralne.

Python

Wysoko poziomowy język programowania ogólnego przeznaczenia. Cechuje go przejrzystośc i zwięzłość. Jest sugerowany dla początkujących. Python rozwija się jako Open Source.
